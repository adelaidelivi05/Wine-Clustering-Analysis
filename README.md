# Wine Clustering Analysis

## Overview

This project applies multiple clustering algorithms to identify patterns within a wine dataset. Different clustering methods were evaluated using clustering performance metrics to determine the most suitable model.

## Tools

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Files

- 📓 Wine_Clustering_Analysis.ipynb
- 📊 wine_dataset.csv
- 📄 Wine_Clustering_Report.pdf

## Methods

- K-Means
- Agglomerative Clustering
- Gaussian Mixture Model (GMM)
- DBSCAN

## Evaluation

- Silhouette Score
- Davies-Bouldin Index
